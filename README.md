# Freedom

![preview](https://user-images.githubusercontent.com/38132413/180621014-9c1bf2ef-0df4-4582-9dbe-aff71ed9ad31.png)

## Build

    vcvarsall x86
    build.bat

## Required Visual Studio Components

* MSVC x64/x86 build tools
* Windows SDK
* .NET Framework SDK

## Features

- Difficulty Changer:
    * Approach Rate (AR)
    * Circle Size (CS)
    * Overall Difficulty (OD)

 - Singletap Relax  
 - HitCircle Aimbot:
    * Cursor Speed

- Misc:
    * Mod Menu Font Size
    * Hide Mod Menu

## Controls

|    Keys     |   Description  |
|:-----------:|:--------------:|
| Right Click |    Settings    |
| F11         |  Hide Mod Menu |
